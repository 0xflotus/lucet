Liblucet: Sandboxing Library for WebAssembly
==========================================

Liblucet is a library for loading, running, and monitoring ahead-of-time compiled WebAssembly modules in lightweight sandboxes. It is intended to work with modules compiled by `lucetc <https://github.com/fastly/isolation/tree/master/lib/lucetc>`_.
